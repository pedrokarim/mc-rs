Key Features

RakNet & Encryption

Full RakNet UDP transport with ECDH P-384 key exchange and AES-256-CFB8 encryption.

World Generation

Perlin noise terrain with 10 biomes, caves, 8 ores, trees, villages, Nether and End dimensions.

ECS Entities

bevy_ecs-powered entities with 5 mob types, 9 AI behaviors, pathfinding, and spawn systems.

Complete Survival

Combat with armor and enchantments, hunger, crafting, furnaces, anvils, and weather cycles.

Plugin System

Rust API with 15 events, WASM runtime (wasmtime), Lua scripting (mlua), and forms UI.

Anti-Cheat

Speed, fly, noclip, reach detection with violation tracking, rate limiting, and auto-kick.

Quick Start

# Clone and build
git clone https://github.com/pedrokarim/mc-rs.git
cd mc-rs
cargo build --release

# Run tests (924 tests)
cargo test

# Start the server
cargo run --release

The server creates a default server.toml on first run. See Configuration for all options.
